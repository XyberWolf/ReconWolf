<h4 align="center">Web Footprinting Tool</h4>

## Installation

```bash
git clone https://github.com/XyberWolf/ReconWolf.git
cd ReconWolf
chmod +x install.sh && ./install.sh
```

## Examples
* For Help
`reconwolf -h`

* For Whois
`reconwolf -w site.com` (or)
`reconwolf --whois=site.com`
